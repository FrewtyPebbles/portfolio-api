## Non-traditional Functionalities

 - Loops

```
loop {
	print("Loop!\n");
}
```

Loops will loop until the path of execution hits a `break` or a `return`.

 - For loops

```
// String Example

for i in "abc" {
	print(add(i, "\n"));
}

// output:
// a
// b
// c

// Integer Example

for i in 3 {
	print(add(i, "\n"));
}

// output:
// 0
// 1
// 2

// List Example

for i in ["1", 2.0, 3] {
	print(add(i, "\n"));
}

// output:
// 1
// 2.0
// 3

// Variable Example

variable = [0, 1, 2];
for i in variable {
	print(add(i, "\n"));
}

// output:
// 0
// 1
// 2
```
